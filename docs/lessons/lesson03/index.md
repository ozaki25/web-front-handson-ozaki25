# lesson03: リストで並べる

## ゴール

このレッスンの **主役はこれ 1 つ** です。

- 順序なしリスト `<ul>` の中に `<li>` を並べて、箇条書きを書ける

主役を身につけたら、次の機能も使えるようになります（本文の **補足** で扱います）。

- 順序付きリスト `<ol>`（手順や順位など、順番に意味があるとき）
- リストの入れ子

**初回は `<ul>` + `<li>` だけ覚えれば十分** です。`<ol>` は「数字を振りたいとき」と覚えて、必要になった時に補足を見直すで OK。

## 解説

### リストは「同じ種類のものを並べる」タグ

箇条書きにしたい項目（好きな食べ物、やりたいこと、手順など）は、ただの段落を並べるのではなく **リストタグ** で書くのが HTML らしい書き方です。

HTML のリストは 2 種類あります。

- `<ul>`: unordered list。**順序なし** のリスト。項目の順番に意味がないときに使う（好きなもの、タグの一覧、メニュー項目など）。
- `<ol>`: ordered list。**順序付き** のリスト。手順や順位など、順番に意味があるときに使う。

どちらも、中に入れる各項目は `<li>`（list item）で囲みます。

### 書き方（順序なしリスト）

```html
<ul>
  <li>りんご</li>
  <li>みかん</li>
  <li>ぶどう</li>
</ul>
```

ブラウザは既定で黒丸（`•`）のマーカーを付けて表示します。

### 補足: 順序付きリスト `<ol>`

順番に意味があるとき（手順 / ランキング）は `<ol>` を使います。`<ul>` を `<ol>` に書き換えるだけで番号が自動で振られます。

```html
<ol>
  <li>卵を割る</li>
  <li>混ぜる</li>
  <li>焼く</li>
</ol>
```

`1.` `2.` `3.` のような番号が自動で付きます。番号を HTML に書き込む必要はありません。ブラウザが順番を数えてくれます。

### 補足: 入れ子のリスト

`<li>` の中にさらに `<ul>` や `<ol>` を入れると、ツリー状のリストになります。

```html
<ul>
  <li>
    果物
    <ul>
      <li>りんご</li>
      <li>みかん</li>
    </ul>
  </li>
  <li>
    野菜
    <ul>
      <li>トマト</li>
      <li>なす</li>
    </ul>
  </li>
</ul>
```

入れ子にするときは、子の `<ul>` を **親の `<li>` の中** に書きます。`<ul>` の直接の子は `<li>` だけにするのが基本ルールです。

### どう使い分けるか

- 順番を変えても意味が変わらない → `<ul>`
- 順番を入れ替えると意味が変わる（手順、ランキングなど）→ `<ol>`

「なんとなく箇条書きにしたい」という場合は、まず `<ul>` を選んでおけば間違いは少ないです。

### 前レッスンの状態から始める

これまでのレッスンで作ったプロジェクトを開き直します。`index.html` は以下の状態になっているはずです。

```html
<!DOCTYPE html>
<html lang="ja">
  <head>
    <meta charset="UTF-8" />
    <title>自己紹介</title>
  </head>
  <body>
    <h1>オザキの自己紹介</h1>

    <h2>ひとこと</h2>
    <p>
      いまは <strong>HTML の基礎</strong> を学んでいます。読むだけでなく、<em>自分でも手を動かして</em> 覚えていきたいです。
    </p>
  </body>
</html>
```

### コピペで動かす

`<body>` の末尾に「好きなもの」のリストと「やりたいこと」のリストを追加します。全体を次のコードに置き換えてください。

```html
<!DOCTYPE html>
<html lang="ja">
  <head>
    <meta charset="UTF-8" />
    <title>自己紹介</title>
  </head>
  <body>
    <h1>オザキの自己紹介</h1>
    <p>はじめまして。Web フロントエンドを勉強中です。</p>
    <h2>ひとこと</h2>
    <p>
      いまは <strong>HTML の基礎</strong> を学んでいます。読むだけでなく、<em>自分でも手を動かして</em> 覚えていきたいです。
    </p>
    <h2>好きなもの</h2>
    <ul>
      <li>コーヒー</li>
      <li>散歩</li>
      <li>本</li>
    </ul>
    <h2>今年やりたいこと</h2>
    <ol>
      <li>Next.js で小さなアプリを作る</li>
      <li>毎週 1 本ブログを書く</li>
      <li>早起きする</li>
    </ol>
  </body>
</html>
```

### 期待出力

- プレビューに「好きなもの」という見出しが出て、その下に黒丸付きで 3 項目が縦に並ぶ。
- 続いて「今年やりたいこと」という見出し、その下に `1.` `2.` `3.` の番号付きで 3 項目が並ぶ。
- DevTools の Elements パネルで見ると、`<ul>` / `<ol>` の中に `<li>` が 3 つずつ入っている構造が確認できる。

### 変えてみる

1. 「好きなもの」の項目を自分の好きなもの 3 つに書き換える。
2. 「今年やりたいこと」を自分の目標 3 つに書き換える。
3. 「好きなもの」の `<ul>` を `<ol>` に書き換えて保存し、見た目が番号付きに変わることを確認する。確認が済んだら `<ul>` に戻す。

### 自分で書く

`<h2>今年やりたいこと</h2>` の上に、新しく「行ったことのある国」というセクションを追加します。見出し + 順序なしリスト（項目 3 つ以上）を、コピペせずに手で書いてみます。

たとえば次のような形になるはずです。完成したら本文のコード例と構造を見比べてください。

```html
<h2>行ったことのある国</h2>
<ul>
  <li>日本</li>
  <li>台湾</li>
  <li>アメリカ</li>
</ul>
```

### 発展（任意）

時間があれば、入れ子のリストも試してみます。「好きなもの」の中の項目を 2 階層にしてみる、など:

<ul>

## まとめ

- 順序に意味がないものは `<ul>`、順序に意味があるものは `<ol>`。各項目は `<li>` で囲む。
- リストは入れ子にできる。子のリストは親の `<li>` の中に置く。
- 自己紹介ページに「好きなもの」と「やりたいこと」のリストが追加された。別のレッスンで、リンクと画像を加えてページらしい見た目に近付けていく。
