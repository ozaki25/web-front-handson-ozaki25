# lesson02: テキストを書く

## ゴール

- 見出し（`<h1>` 〜 `<h6>`）と段落（`<p>`）を使い分けて文章を構造化できる。
- 強い意味（`<strong>`）と軽い強調（`<em>`）を文章中で使える。
- 自己紹介ページの冒頭（名前・簡単な紹介文・キャッチコピー）を HTML で書ける。

## 解説

### 見出しはページの「目次」

見出しは、ページの中で「ここから新しい話題」と示すためのタグです。`<h1>` から `<h6>` まで 6 段階あり、数字が小さいほど重要度が高くなります。

```html
<h1>いちばん大きな見出し（ページの主題）</h1>
<h2>次の見出し（セクション）</h2>
<h3>その下（サブセクション）</h3>
```

ブラウザは見出しの文字を大きく太く表示しますが、本質は「文字の大きさ」ではなく **「意味の階層」** です。見出しは、検索エンジン、スクリーンリーダー（画面の内容を読み上げる支援技術）、DevTools のアウトラインなど、見た目以外にも使われます。

階層のルールはシンプルです。

- `<h1>` は原則としてページに 1 つ（そのページの主題を示す）。
- `<h2>` 以降は上から順に使う。`<h1>` の次に `<h3>` を置いて `<h2>` を飛ばす、のようなジャンプは避ける。

見た目を整えたいだけの理由で見出しの段階を選ばないように注意します。見た目は後のレッスンで CSS を使って調整するので、いまは **意味に合う段階を選ぶ** ことに集中します。

### 段落は `<p>`

ふつうの本文は `<p>` で囲みます。paragraph（段落）の意味です。HTML では、改行（Enter）を入れても画面上では改行になりません。改行を見せたい場合は段落を分けるのが基本です。

```html
<p>こんにちは。はじめまして。</p>
<p>これは 2 つ目の段落です。</p>
```

### 強調には `<strong>` と `<em>`

文章の一部に意味の重みを付けたいときは、次の 2 つを使います。

- `<strong>`: 強い重要度を表す。ブラウザは既定で **太字** で表示する。
- `<em>`: 軽い強調（emphasis）。ブラウザは既定で _斜体_ で表示する。日本語フォントでは斜体の効果が薄いことが多いが、意味付けとしては有効。

```html
<p>締切は <strong>明日</strong> です。</p>
<p>この部分を <em>少しだけ</em> 強く読みたい。</p>
```

見た目を太字にしたいだけなら CSS の出番です。`<strong>` / `<em>` は「ここは意味として重要・強調」というときに使います。

## 演習

### 途中から始める場合

これまでのレッスンで作った `index.html` を続けて使うのが理想ですが、手元に無ければ、新規 StackBlitz の Vanilla（HTML / CSS / JS）テンプレート（<https://stackblitz.com/edit/web-platform>）を開き、下の「出発点のコード」をそのまま貼って始めてください。

<details>
<summary>出発点のコード</summary>

**`index.html`**

```html
<!DOCTYPE html>
<html lang="ja">
  <head>
    <meta charset="UTF-8" />
    <title>はじめての HTML</title>
  </head>
  <body>
    <h1>Hello, HTML</h1>
    <p>これは最初のページです。</p>
  </body>
</html>
```

</details>

### 前レッスンの成果物を開き直す

これまでのレッスンで作った StackBlitz のプロジェクトを開きます。もし閉じてしまっていたら、同じように HTML / CSS / JS の Vanilla プロジェクトを作り、`index.html` を次の状態にしておきます。

```html
<!DOCTYPE html>
<html lang="ja">
  <head>
    <meta charset="UTF-8" />
    <title>はじめての HTML</title>
  </head>
  <body>
    <h1>Hello, HTML</h1>
    <p>これは最初のページです。</p>
  </body>
</html>
```

### コピペで動かす

`<body>` の中身を次の内容に置き換えます。`<title>` も「自己紹介」に変えます。

```html
<!DOCTYPE html>
<html lang="ja">
  <head>
    <meta charset="UTF-8" />
    <title>自己紹介</title>
  </head>
  <body>
    <h1>オザキの自己紹介</h1>
    <p>はじめまして。Web フロントエンドを勉強中です。</p>
    <h2>ひとこと</h2>
    <p>
      いまは <strong>HTML の基礎</strong> を学んでいます。読むだけでなく、<em>自分でも手を動かして</em> 覚えていきたいです。
    </p>
  </body>
</html>
```

保存するとプレビューが自動更新されます。

### 期待出力

- プレビューの一番上に大きな文字で「オザキの自己紹介」。
- その下に「はじめまして。Web フロントエンドを勉強中です。」の段落。
- さらに下に「ひとこと」という少し小さい見出し。
- 最後の段落の中で、「HTML の基礎」の部分が太字、「自分でも手を動かして」の部分が斜体（日本語フォントでは傾かない場合もあるが、DevTools で確認すると `<em>` タグで囲まれている）。
- ブラウザのタブ名が「自己紹介」。

DevTools の Elements パネルでツリーを開き、`<h1>` / `<h2>` / `<p>` / `<strong>` / `<em>` の入れ子関係を眺めてみます。`<p>` の中に `<strong>` と `<em>` が入っていることが確認できるはずです。

### 変えてみる

1. `<h1>` の「オザキ」を自分の名前に書き換える。
2. 最初の `<p>` の紹介文を、自分の状況に合わせた 1 文に書き換える（「〇〇大学で勉強中です」など）。
3. 2 つ目の `<p>` の強調箇所を、自分の学びたいこと（`<strong>`）と気持ち（`<em>`）に置き換える。

### 自分で書く

`<h2>ひとこと</h2>` の下に、新しく `<h2>このページについて</h2>` という見出しを加え、その下に段落を 2 つ書き足してみます。例:

```html
<h2>このページについて</h2>
<p>このページは Web フロントエンドの練習のために作っています。</p>
<p>少しずつ内容を増やしていきます。</p>
```

コピペせず、タグの打ち込みから自分で行うのがおすすめです。打ち間違えたときにブラウザがどう表示するか（開始タグだけで終了タグを忘れるとどうなるか、など）を観察すると、タグの意味が体で分かってきます。

## まとめ

- 見出し `<h1>` 〜 `<h6>` は「意味の階層」。段を飛ばさず、上から順に使う。`<h1>` はページに 1 つが基本。
- 段落は `<p>`。HTML では改行しただけでは画面上の改行にならない。
- `<strong>` は強い重要度、`<em>` は軽い強調。見た目目的ではなく意味付けのタグ。
- 自己紹介ページの「名前 + 紹介文 + ひとこと」の骨格が出来た。次レッスンでは「好きなもの」のリストを追加する。
