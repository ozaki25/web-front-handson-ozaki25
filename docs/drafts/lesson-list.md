# レッスン一覧（ドラフト v1）

全体像と各レッスンの射程をまとめた作業用ドキュメント。各レッスンの本文を書く前に、順序・粒度・抜け漏れをレビューするためのもの。

## 設計方針

- 読者: Web 開発の初心者。HTML / CSS / JS / React / Next.js をほぼ触ったことがない。
- 1 レッスンで新しく導入する概念は原則 1 つ。
- 座学 → 演習 の順。StackBlitz で即動かす。
- 最終到達点: Next.js (App Router) で「フォーム + データ表示 + ルーティング」の小さなアプリを自作できる。
- 各レッスンはおおよそ 20〜40 分想定（読む + 手を動かす）。

## 章構成

| 章 | テーマ | レッスン数 | 範囲 |
|---|---|---|---|
| 1 | HTML / CSS | 10 | lesson01 〜 lesson10 |
| 2 | JavaScript | 10 | lesson11 〜 lesson20 |
| 3 | TypeScript | 5 | lesson21 〜 lesson25 |
| 4 | React | 10 | lesson26 〜 lesson35 |
| 5 | Next.js | 10 | lesson36 〜 lesson45 |

合計 **45 レッスン**。

## 章 1: HTML / CSS

### lesson01: HTML ってなに？

- **ゴール**: Web ページが HTML でできていることを理解し、最小の HTML を書いてブラウザで表示できる。
- **新概念**: HTML、タグ、`<!DOCTYPE html>` / `<html>` / `<head>` / `<body>`。
- **演習**: StackBlitz で「Hello, HTML」と表示する 1 ページを作る。

### lesson02: テキストを書く

- **ゴール**: 見出し、段落、強調を使い分けて文章を構造化できる。
- **新概念**: `<h1>〜<h6>` / `<p>` / `<strong>` / `<em>`。
- **演習**: 自己紹介ページを作る（見出し + 段落 + 強調）。

### lesson03: リストで情報を並べる

- **ゴール**: 順序付きリスト / 順序なしリストを書ける。
- **新概念**: `<ul>` / `<ol>` / `<li>`。
- **演習**: 好きなもの一覧と「やることリスト」を作る。

### lesson04: リンクと画像

- **ゴール**: 他ページへのリンクと画像を埋め込める。
- **新概念**: `<a href>` / `<img src alt>`。
- **演習**: 自己紹介ページに外部リンクと画像を追加する。

### lesson05: ページを構造化する

- **ゴール**: セマンティックタグを使ってページの骨格を組める。
- **新概念**: `<header>` / `<nav>` / `<main>` / `<section>` / `<footer>`。
- **演習**: これまでに書いたページを構造タグで整理する。

### lesson06: フォームを作る

- **ゴール**: 入力欄と送信ボタンを持つフォームを書ける。
- **新概念**: `<form>` / `<input type>` / `<label>` / `<button>`。
- **演習**: 名前とメッセージを受け取るお問い合わせフォームを作る（送信先は後のレッスンで）。

### lesson07: CSS ってなに？

- **ゴール**: CSS の役割と、HTML に CSS を適用する 3 つの方法を理解する。セレクタの基本を書ける。
- **新概念**: `<style>` タグ / `<link rel="stylesheet">` / 要素セレクタ / クラスセレクタ / `class` 属性。
- **演習**: 自己紹介ページの見出しに色を付ける。

### lesson08: 色と文字を整える

- **ゴール**: 色、背景色、フォントサイズ、行間を指定できる。
- **新概念**: `color` / `background-color` / `font-size` / `line-height` / `font-family`。
- **演習**: 自己紹介ページの見た目を整える。

### lesson09: ボックスモデルと余白

- **ゴール**: 余白・枠・サイズの指定を使い分けられる。
- **新概念**: `margin` / `padding` / `border` / `width` / ボックスモデル。
- **演習**: カード風の見た目を作る。

### lesson10: Flexbox で並べる

- **ゴール**: Flexbox で要素を横並び・縦並びにできる。
- **新概念**: `display: flex` / `flex-direction` / `gap` / `justify-content` / `align-items`。
- **演習**: ナビゲーションメニューとカード 3 枚を横並びにする。

## 章 2: JavaScript

### lesson11: 最初の JavaScript

- **ゴール**: HTML に JS を読み込み、コンソールに出力できる。変数を宣言できる。
- **新概念**: `<script>` / `console.log` / `let` / `const`。
- **演習**: 自分の名前を変数に入れてコンソールに表示する。

### lesson12: 値の種類

- **ゴール**: JS の基本的な値の種類を判別できる。
- **新概念**: 文字列 / 数値 / 真偽値 / `null` / `undefined` / テンプレートリテラル。
- **演習**: いくつかの値を宣言し、`typeof` で種類を確認する。

### lesson13: 条件で分岐する

- **ゴール**: 条件に応じて処理を分岐できる。
- **新概念**: `if` / `else if` / `else` / 比較演算子 / 論理演算子。
- **演習**: 年齢を入れると「成人 / 未成年」を表示するコードを書く。

### lesson14: 配列を扱う

- **ゴール**: 配列を作り、要素を参照・追加・書き換えできる。
- **新概念**: `[]` / インデックス / `length` / `push` / `pop`。
- **演習**: やることリスト（配列）を作り、要素を追加して出力する。

### lesson15: 繰り返し処理

- **ゴール**: 配列の全要素を処理できる。
- **新概念**: `for...of` / `for` / `forEach`。
- **演習**: 配列の全要素をコンソールに表示する。

### lesson16: 関数

- **ゴール**: 関数を定義して呼び出せる。引数と戻り値を使える。
- **新概念**: `function` 宣言 / アロー関数 / `return`。
- **演習**: 2 つの数の合計を返す関数を書く。

### lesson17: オブジェクト

- **ゴール**: オブジェクトを作り、プロパティにアクセスできる。
- **新概念**: `{ key: value }` / ドット記法 / ブラケット記法 / プロパティの追加。
- **演習**: `user` オブジェクトを作り、名前と年齢を出力する。

### lesson18: 配列の便利メソッド

- **ゴール**: 配列を変換・絞り込みできる。
- **新概念**: `map` / `filter` / `find`。
- **演習**: ユーザー配列から名前だけの配列を作る、成人だけに絞る。

### lesson19: DOM を操作する

- **ゴール**: JS から HTML の内容を取得・書き換えできる。
- **新概念**: `document.querySelector` / `textContent` / `innerHTML`。
- **演習**: ボタンクリックで見出しの文字を変える（`onclick` は次レッスン、ここではコンソールからの操作）。

### lesson20: イベントで画面を動かす

- **ゴール**: クリックや入力に反応して画面を変更できる。
- **新概念**: `addEventListener` / `click` イベント / `input` イベント / `event.target.value`。
- **演習**: カウンターアプリ（ボタンで数字が増える）と、入力欄の値をリアルタイム表示するページを作る。

## 章 3: TypeScript

### lesson21: TypeScript ってなに？

- **ゴール**: 型がなぜ必要かを理解し、変数に型を付けられる。
- **新概念**: 静的型付け / 型注釈 / `string` / `number` / `boolean`。
- **演習**: 型注釈付きで変数を宣言し、間違った値を入れてエラーを確認する。

### lesson22: 関数の型

- **ゴール**: 関数の引数と戻り値に型を付けられる。
- **新概念**: 引数の型注釈 / 戻り値の型注釈 / `void`。
- **演習**: 合計関数に型を付ける。

### lesson23: オブジェクトの型

- **ゴール**: オブジェクトの形を型で表現できる。
- **新概念**: インラインオブジェクト型 / `type` エイリアス / オプショナルプロパティ。
- **演習**: `User` 型を定義し、関数の引数として使う。

### lesson24: 配列とユニオン型

- **ゴール**: 配列の型、複数の型のどれか、を表現できる。
- **新概念**: `T[]` / `Array<T>` / ユニオン型 (`|`) / リテラル型。
- **演習**: `"admin" | "guest"` の role を持つユーザー配列を定義する。

### lesson25: React で使う TypeScript

- **ゴール**: React コンポーネントで props に型を付ける準備をする。
- **新概念**: コンポーネントの props 型 / `ReactNode` のさわり。
- **演習**: 型だけ先に書いてみる（実装は章 4 で）。

## 章 4: React

### lesson26: React ってなに？

- **ゴール**: React の役割を理解し、StackBlitz で最小の React プロジェクトを起動できる。
- **新概念**: React / SPA / コンポーネント / Vite。
- **演習**: StackBlitz のテンプレートから起動して「Hello, React」と表示する。

### lesson27: JSX を書く

- **ゴール**: JSX の書き方と、JS の式を埋め込む方法を理解する。
- **新概念**: JSX / `{式}` / `className` / 属性のキャメルケース / フラグメント。
- **演習**: 現在時刻を埋め込んで表示するコンポーネントを書く。

### lesson28: コンポーネントと props

- **ゴール**: コンポーネントを分けて再利用でき、props で値を受け渡せる。
- **新概念**: コンポーネントの分割 / props / 型付き props。
- **演習**: `Greeting` コンポーネントに名前を渡して 3 パターン表示する。

### lesson29: 配列を描画する

- **ゴール**: 配列データをコンポーネントのリストとして描画できる。
- **新概念**: `.map` でのレンダリング / `key` prop。
- **演習**: ユーザー一覧を表示する。

### lesson30: state で状態を持つ

- **ゴール**: `useState` で状態を持ち、再描画の仕組みを理解する。
- **新概念**: `useState` / 状態更新関数 / 再レンダリング。
- **演習**: カウンターアプリを React で作る。

### lesson31: イベントを扱う

- **ゴール**: ボタンクリックなどのイベントで state を更新できる。
- **新概念**: `onClick` / イベントハンドラの型。
- **演習**: カウンターに「リセット」「マイナス」ボタンを追加する。

### lesson32: フォームを扱う

- **ゴール**: 入力欄の値を state と同期させて制御できる。
- **新概念**: 制御コンポーネント / `onChange` / `value`。
- **演習**: 入力内容のプレビューが出るフォームを作る。

### lesson33: 条件で出し分ける

- **ゴール**: 条件によって別の要素を描画できる。
- **新概念**: `&&` / 三項演算子 / 早期 return。
- **演習**: ログイン状態の切り替えで表示を変える。

### lesson34: useEffect で副作用

- **ゴール**: マウント時に実行される副作用を書ける。
- **新概念**: `useEffect` / 依存配列 / クリーンアップ関数。
- **演習**: マウント時にダミー API を叩いて結果を表示する。

### lesson35: 親子コンポーネントの連携

- **ゴール**: 子から親に値を伝える（コールバック props）を書ける。
- **新概念**: コールバック props / 状態の持ち上げ（state lifting）。
- **演習**: TODO 追加フォーム（子）と一覧（親）を連携させる。

## 章 5: Next.js

### lesson36: Next.js ってなに？

- **ゴール**: Next.js と App Router の役割を理解し、StackBlitz で最小のプロジェクトを起動できる。
- **新概念**: Next.js / App Router / `app/page.tsx`。
- **演習**: StackBlitz のテンプレートから起動してトップページを表示する。

### lesson37: ページを増やす

- **ゴール**: ファイルベースルーティングで新しいページを追加できる。
- **新概念**: `app/xxx/page.tsx` / URL とファイル構造の対応。
- **演習**: `/about` ページと `/contact` ページを追加する。

### lesson38: リンクと共通レイアウト

- **ゴール**: `<Link>` でページ間を移動でき、`layout.tsx` で共通部を作れる。
- **新概念**: `next/link` / `layout.tsx` / `children`。
- **演習**: ヘッダーとフッターを全ページ共通にする。

### lesson39: Server Component と Client Component

- **ゴール**: 2 種類のコンポーネントの違いを理解し、使い分けられる。
- **新概念**: Server Component（デフォルト）/ Client Component / `"use client"`。
- **演習**: カウンターを Client Component として分離する。

### lesson40: データを取得する

- **ゴール**: Server Component で外部 API のデータを取得して描画できる。
- **新概念**: Server Component での `fetch` / async コンポーネント。
- **演習**: ダミー API から記事一覧を取得して表示する。

### lesson41: 動的ルート

- **ゴール**: URL の一部をパラメータとして受け取るページを作れる。
- **新概念**: `[slug]` ディレクトリ / `params`。
- **演習**: `/posts/[id]` ページで記事詳細を表示する。

### lesson42: メタデータを設定する

- **ゴール**: ページのタイトルや説明を設定できる。
- **新概念**: `export const metadata` / タイトル / OG。
- **演習**: 各ページにふさわしいタイトルと説明を設定する。

### lesson43: フォームと Server Actions

- **ゴール**: フォーム送信をサーバー側で処理できる。
- **新概念**: Server Actions / `action` 属性に関数を渡す / `revalidatePath`。
- **演習**: TODO 追加フォームを Server Actions で実装する（メモリ上の配列に保存）。

### lesson44: 小さなアプリを作る（統合演習）

- **ゴール**: これまでの知識を使って「フォーム + 一覧 + 詳細ページ」を持つアプリを作る。
- **新概念**: なし（統合）。
- **演習**: 簡単な掲示板アプリ（投稿フォーム / 一覧 / 詳細ページ / メタデータ）を実装する。

### lesson45: Vercel にデプロイする

- **ゴール**: 作ったアプリを Vercel で公開できる。
- **新概念**: Git 連携 / Vercel のプロジェクト作成 / 環境変数（触りだけ）。
- **演習**: GitHub にプッシュして Vercel で公開し、URL を共有できる状態にする。
