# Web フロントエンド入門

Web フロントエンドをこれから学ぶ人が、手を動かしながら基礎をゼロから積み上げられる学習コンテンツです。

はじめての方は[はじめに](/introduction/)を読んでください。

## カリキュラム

1. Web の素材（HTML / CSS）
2. 動きをつける（JavaScript / DOM）
3. 型と道具（TypeScript / Vite / npm）
4. React
5. Next.js（App Router）
6. 仕上げ: 小さなアプリを作る

## 進捗
