# はじめに

## このコンテンツについて

Web フロントエンドをこれから学ぶ人向けの学習コンテンツです。HTML / CSS / JavaScript から React、Next.js まで、基礎をゼロから積み上げていきます。

座学で仕組みや背景を押さえ、要所でコードを書いて手触りを確認する構成です。読み進めるだけでも理解できますし、実際に手を動かすことでより深く定着させることもできます。

## 進め方

- レッスンは番号順に並んでいます。基礎から順に積み上げる構成なので、上から順に進めてください
- 1 日 1 レッスンに縛る必要はありません。集中できる日は一気に複数レッスン進めても構いません
- コードを動かす場面では StackBlitz を使います。ブラウザだけで動くので、準備は不要です（GitHub ログインすると保存・Fork が使えて便利です）
- 暗記しようとしなくて大丈夫です。何度も出てくる概念は、その都度該当レッスンに戻って確認してください

## 完走したとき

Next.js（App Router）で小さなアプリを自分で作れる状態を目指します。

- フォームを置いたページ
- データを一覧表示するページ
- ルーティングのあるページ
